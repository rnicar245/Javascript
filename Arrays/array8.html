Array.forEach() recorre cada elemento del array, invocando una función por cada uno que encuentra.
Ejemplo

array = [1,2,3,4];
array.forEach(function(element,index){
console.log("array["+indice+"] = "+elemento+");
});

Resultado

Array.every() recorre cada elemento del array y devuelve true si todos los elementos cumplen la condición de la función que se le pasa por parámetro.
Ejemplo

array = [1,2,3,"cuatro"];
array.every(function(elemento){
return !isNaN(elemento);
});

Resultado

Array.some() recorre cada elemento del array y devuelve true al menos uno de los elementos cumplen la condición de la función que se le pasa por parámetro.
Ejemplo

array = [1,2,3,"cuatro"];
array.some(function(elemento){
return !isNaN(elemento);
});

Resultado
